# Brand Deal Auditor

A privacy-first, AI-powered audit tool for creators to analyze brand contracts, identify risks, and negotiate professional terms.

## Project Overview
Brand Deal Auditor helps influencers and creators protect their business. By leveraging AI to scan contracts for "gotchas" and predatory clauses, the platform provides plain-English translations, risk assessments, and counter-offer generation in seconds.

## Key Features
*   **Privacy-First AI Analysis:** Processes contracts in-memory; no raw documents are ever stored.
*   **Plain-English Translations:** Breaks down complex legalese into actionable insights.
*   **Deal Health Scoring:** Instant 0–100 risk assessment for every contract.
*   **Counter-Offer Generator:** Customizes responses in multiple tones (Firm, Friendly, Casual).
*   **Subscription Tiers:** Flexible credit-based system to support both growing creators and power users.
*   **Secure History:** Searchable history of contract summaries and negotiation threads.

## Tech Stack
*   **Frontend:** React / Next.js
*   **Styling:** Tailwind CSS, GSAP (animations)
*   **Backend/Database:** Supabase (PostgreSQL & Edge Functions)
*   **Authentication:** Supabase Auth (Google/Apple)
*   **AI Framework:** Antigravity
*   **Deployment:** Vercel
